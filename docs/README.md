# Xuất bản portfolio bằng GitHub Pages

Thư mục này đã sẵn sàng để đưa lên static hosting. Website không cần cài đặt hoặc build.

1. Tạo repository công khai tên `portfolio-thao-ngan` trên GitHub.
2. Tải toàn bộ nội dung trong thư mục này lên nhánh `main`.
3. Vào **Settings → Pages**.
4. Chọn **Deploy from a branch**, nhánh `main`, thư mục `/ (root)` rồi lưu.
5. URL công khai hiện tại: `https://quytranngoc34-ai.github.io/portfolio/`.

## Trước khi xuất bản cuối

- Thay hai trạng thái “Minh chứng đang cập nhật” của Bài 5 và Bài 6 khi có link.
- PDF hiện đã chứa URL và QR trỏ tới website công khai.
- Có thể thay monogram `TN` bằng ảnh chân dung thật sau này mà không cần đổi cấu trúc trang.

PDF đã nằm cùng website với tên `vo-thi-thao-ngan-portfolio.pdf`; các nút **Xem PDF** và **Tải PDF** hoạt động ngay sau khi deploy.
